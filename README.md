# Portfólio — Vitor Pietro

Portfólio pessoal de Vitor Pietro de Souza Ferreira, estudante de Análise e Desenvolvimento de Sistemas e desenvolvedor web em formação.

O site apresenta experiência prática com sistemas administrativos, rotinas comerciais, JavaScript, SQL, Supabase, Node.js e suporte técnico. O objetivo é facilitar a avaliação do perfil para oportunidades iniciais, estágio e posições de desenvolvedor júnior ou auxiliar de TI.

## Tecnologias

- HTML5 semântico
- CSS3 responsivo
- JavaScript
- Node.js para servidor local e verificações
- Git e GitHub

O site é feito em HTML, CSS e JavaScript puro, sem frameworks ou bibliotecas de animação. As animações, microinterações e efeitos (partículas no hero, reveal no scroll, glow seguindo o cursor, tilt nos cards e barra de progresso) são implementados com CSS moderno e JavaScript leve, respeitando `prefers-reduced-motion` e com efeitos pesados desativados em telas touch.

## Como executar localmente

Requisitos: Node.js 18 ou superior e npm.

```bash
npm install
npm run dev
```

Abra `http://127.0.0.1:5055` no navegador.

Para executar as verificações do projeto:

```bash
npm run check
```
